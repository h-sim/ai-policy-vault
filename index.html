ノイズ最小（MVP） RSS配信

AI Change Watcher

AIプラットフォーム（OpenAI など）の変更を監視し、読む価値のある変更だけを優先して RSS で配信します。

Important を購読（推奨） All を購読（参考） 一覧を見る（Changes） GitHub

Important は Breaking/High 中心、All は広め（ノイズ混在の可能性あり）。

Important / All の違い

  • Important：行動につながる変更を優先（ノイズ抑制）。
  • All：検知した変更を広めに掲載（ノイズが混ざる場合あり）。

RSS のウィンドウ更新で「古い項目が落ちるだけ」など、価値が低い変化は抑制します。

監視対象（MVP）

  • OpenAI Developer Changelog RSS
  • OpenAI News RSS
  • OpenAI OpenAPI Spec（YAML）

使い方（購読）

  1. Feedly / Inoreader / NetNewsWire などの RSS リーダーを用意
  2. feed.xml（Important）を追加
  3. 必要なら feed_all.xml（All）も追加

免責

  • 本フィードは「変更の検知・要約」を提供するもので、正確性を保証しません。
  • 重要な判断は必ず一次情報（公式発表・原文）を確認してください。

Pro（予定）

要望を募集中です（まずは見込み客回収）。

  • メール通知 / Slack / Discord / Webhook
  • 日本語の短い影響サマリ
  • 週次まとめ / 過去検索

興味があれば GitHub Issues に pro-interest を付けて投稿（1行でOK）。

問い合わせ / フィードバック

「これはノイズ / 重要」など、Important の品質調整に使います。
